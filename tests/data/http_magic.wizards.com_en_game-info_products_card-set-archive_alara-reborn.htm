Alara Reborn is a 145-card set featuring the new Cascade mechanic, multicolored equipment, and a twist on cycling that will help you dig through your deck for answers to dangerous new threats.

Set Name: Alara Reborn
Block: Set 3 of 3 of the Shards of Alara Block
Number of Cards: 145

Languages: English, German, French, Italian, Japanese, Portuguese (Brazil), Russian, Chinese Simplified, Spanish
